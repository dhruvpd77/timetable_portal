Instructions for Excel Format
  • Time Slot Format:
    Each timing cell should be written exactly like:
    8:45 to 9:45 9:45 to 10:45 11:30 to 12:30 12:30 to 13:30 13:45 to 14:45
  • Assignment (Entry) Format:
    For each cell (for a batch/time/slot), use:
    FacultyName-SubjectName-RoomOrLab
    Examples:
    • MGV-DM-302 (Faculty: MGV, Subject: DM, Room: 302)
    • KMS-PY_2-406-2(L) (Faculty: KMS, Subject: PY_2, Lab: 406-2, mark lab with (L))
    Use (L) after Room/Lab code to indicate a Lab. Room and Lab naming should match database entries exactly.
  • Batches: Your header (top row after "Day" and "Timing") should include batch names like A1, A2, B1, B2 etc.
    Only columns with letters as prefix and digits as suffix (e.g. A1, B3, C12) will be detected as batches.
